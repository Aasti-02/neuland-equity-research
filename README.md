# Equity Research & Valuation Analysis – Neuland Laboratories Ltd.

A comprehensive equity research report on **Neuland Laboratories Ltd.**, covering business analysis, industry outlook, financial statement analysis, ratio analysis, peer comparison, and intrinsic valuation using the Discounted Cash Flow (DCF) method.

---

## Project Overview

This project aims to evaluate the intrinsic value of Neuland Laboratories Ltd. by combining fundamental analysis with financial modeling.

The report includes:

- Company Overview
- Industry & Business Analysis
- Historical Financial Performance
- Financial Ratio Analysis
- Peer Comparison
- DCF Valuation
- Sensitivity Analysis
- Investment Thesis
- Risk Assessment

---

## Key Highlights

- Built a 5-year DCF valuation model
- Performed revenue, profitability and cash flow analysis
- Evaluated ROE, ROA, ROCE and leverage trends
- Conducted peer comparison with listed pharmaceutical companies
- Developed bear, base and bull case valuation scenarios
- Prepared an investment recommendation supported by valuation analysis
